## Installation

Add the NuGet packages to your WPF application or library. From the Package Manager Console type

```powershell
Install-Package NWaveform.WPF
```

For [VLC](http://www.videolan.org/) support, add

```powershell
Install-Package NWaveform.Vlc
```
